# Functional Specifications

One file per feature area. To add a feature, create a new file here using the template in `../CONVENTIONS.md` and add a row to the table below.

| Area | File | Summary |
|---|---|---|
| Items | [items.md](./items.md) | Create, read, update, delete items; quick-add; quantity operations. |
| Item Types | [item-types.md](./item-types.md) | Manage categories; merge; reassign-on-delete. |
| Storage Locations | [storage-locations.md](./storage-locations.md) | Manage locations from list or floor plan. |
| Floor Plan | [floor-plan.md](./floor-plan.md) | View, pan, zoom, edit the plan; rooms and shelves. |
| Search | [search.md](./search.md) | Full-text + filters across inventory. |
| Expiration alerts | [expiration-alerts.md](./expiration-alerts.md) | Expiring-soon window; dashboard widget; per-item badges. |
| Low-stock alerts | [low-stock-alerts.md](./low-stock-alerts.md) | Threshold-based alerts at item and type level. |
| Shopping list | [shopping-list.md](./shopping-list.md) | Auto-composed list + manual entries + restock flow. |
| Photos | [photos.md](./photos.md) | Upload, display, delete item photos. |
| Backup & restore | [backup-restore.md](./backup-restore.md) | Daily backups, 30-day retention, manual export/restore. |
